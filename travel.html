Why Boyer-Moore?

Efficient Place Matching: It helps search efficiently for place names that match or contain parts of the input text. So, as a user types, it quickly finds relevant places in a large dataset.

Avoiding Full List Searches: Unlike linear searches, Boyer-Moore doesn't check every character of the dataset.
